## Debugging Methodology

Every audit follows a structured investigation workflow:

1. Observe the issue
2. Reproduce the behavior
3. Inspect Data Layer
4. Validate GTM Triggers
5. Validate Tags
6. Verify Network Requests
7. Confirm GA4 Collection
8. Document Root Cause
9. Validate Fix
